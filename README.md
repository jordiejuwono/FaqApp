# Simple FAQ App
Simple CRUD (Create, Read, Update, and Delete) FAQ App.

## **FAQ App Features**

### **Finished Features**
```
- Login to application to access FAQ features
- Create new FAQ
- Showing FAQ list and show details of selected FAQ
- Updating FAQ details
- Delete selected FAQ
- Showing profile page
- Logout
```
